Privacy Policy

Last updated: May 11, 2026

Overview

Email Manager Pro is a desktop email triage application. It stores mailbox data locally and only sends data to third-party services when the user explicitly enables features that require them.

Data stored locally

The app may store the following data on the user's device:

  • Connected mailbox account metadata.
  • Email metadata and message content retrieved into the local client.
  • Attachments downloaded into the local app database.
  • Local app preferences.
  • Diagnostic events used for support and troubleshooting.

Gmail and Google user data

If the user connects a Gmail account, Email Manager Pro requests access only after the user starts the Google sign-in flow and grants permission. Gmail data is used to display the user's mailbox inside the desktop app, detect important messages, show notifications, manage attachments, and allow user-initiated replies.

Email Manager Pro does not sell Gmail data, does not use Gmail data for advertising, and does not transfer Gmail data to third parties except as necessary to provide user-enabled features.

Secrets and credentials

Sensitive secrets are not intended to be stored as plain values in regular app settings files. The app uses operating-system backed secret storage where supported for values such as mailbox secrets, OAuth tokens, AI provider API keys, and Telegram bot tokens.

When data leaves the device

User data leaves the device only when the user enables features that require external services, such as:

  • OAuth: provider login and token exchange occur with the relevant mailbox provider.
  • Cloud AI: email content relevant to the analysis request may be sent to the configured AI provider.
  • Telegram: notification text and reply workflow content may be sent to Telegram when enabled.

Local AI behavior

When Local AI is active, the app is intended to process analysis on-device. macOS uses the local MLX path and Windows uses the local Ollama path. External cloud behavior only applies if the user separately enables cloud features.

Support reports

The app can export a support report for diagnostics. The report may include app and platform information, mailbox status summaries, recent diagnostic events, and redacted AI settings. It is designed to avoid exporting sensitive secrets such as API keys or Telegram bot tokens.

User choices

Users control whether to connect or remove mailbox accounts and whether to enable Cloud AI, Local AI, or Telegram notifications.
